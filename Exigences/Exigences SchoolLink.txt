Pour que SchoolLink devienne une véritable plateforme SaaS pour les écoles et universités africaines, il faut définir les exigences métier avant même de développer les fonctionnalités.

1. Présentation du projet

SchoolLink est une plateforme web et mobile permettant aux établissements scolaires de gérer :

Les élèves et étudiants
Les enseignants
Les classes
Les présences
Les paiements
Les notes
Les communications
Les statistiques de l'établissement

12. Gestion multi-écoles (SaaS)
Exigence critique

Une école ne doit jamais voir les données d'une autre école.

Chaque établissement possède :

Ses utilisateurs
Ses élèves
Ses enseignants
Ses paiements
Ses statistiques

Architecture :

SchoolLink
 ├─ École A
 ├─ École B
 ├─ École C
 └─ Université D
