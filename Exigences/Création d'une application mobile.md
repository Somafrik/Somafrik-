Création d'une application mobile "SchoolLink"



Ce que mon application mobile peut résoudre



Les problèmes fréquents dans beaucoup d’établissements africains :



* absence de plateforme numérique centralisée
* communication difficile entre école et parents
* gestion manuelle des notes et présences
* paiements scolaires compliqués
* manque d’accès aux cours numériques
* mauvaise organisation administrative
* faible accès aux outils e-learning





Fonctionnalités :



* connexion administrateur / enseignant / étudiant / parent
* gestion des élèves
* notes et bulletins
* absences
* emploi du temps
* notifications
* paiement des frais scolaires
* message école-parent, parent-école, parent-prof, école-prof
* téléchargement bulletins PDF si le parent est insolvable
* mode hors ligne
